## Wiring Diagram

Raspberry Pi Zero WH + Enviro+ Board
(Enviro+ sits directly on the Pi’s 40-pin GPIO header)

PMS5003 Particulate Matter Sensor
(uses 8-pin JST cable → breakout on Enviro+)

Connections:
--------------------------------------------
PMS5003 Pin   →   Enviro+/Pi GPIO Pin
--------------------------------------------
VCC (5V)      →   5V (Pin 2 or 4)
GND           →   GND (Pin 6)
TX            →   Pi RX (GPIO15, Pin 10)
RX            →   Pi TX (GPIO14, Pin 8)
RESET         →   (not connected, optional)
SET           →   (not connected, defaults to active mode)
